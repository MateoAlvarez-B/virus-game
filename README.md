# Virus! – Versión Terminal en Python

Juego inspirado en el clásico **Virus!**, implementado en Python para jugar desde la terminal.

## Características

- 1 jugador vs IA
- Todas las cartas del juego original
- Órganos, virus, medicinas e inmunizaciones
- Cartas especiales fieles al juego real
- Emojis y colores para una experiencia visual más agradable
- Arquitectura modular y profesional
- Tests unitarios con pytest

## Estructura del proyecto

- `src/` → Código fuente
- `tests/` → Tests unitarios
- `README.md` → Documentación
- `pytest.ini` → Configuración de pytest

## Cómo ejecutar

```bash
python -m src.main
